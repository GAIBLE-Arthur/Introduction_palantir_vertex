# Introduction_palantir_vertex
Introduction : Palantir Vertex

Ce use-case concerne l'utilisation de la plateforme Palantir Foundry, les pipeline de données, l'ontologie et l'outil de visualisation par graphes Vertex.

Le but de l'exercice est de démontrer l'utilisation de la plateforme dans le cadre de services de renseignement.

Les données sont générées par Gemini 3, sont fictives et ne proviennent pas d'entité liées comme clientes ou partenaire de Palantir Technologies.
